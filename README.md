# 🤖 AI Financial Assistant (RAG-Based GenAI Application)

An advanced conversational RAG (Retrieval-Augmented Generation) application built using LangChain, FAISS, Groq LLMs, and Streamlit.

## 🚀 Features

- Dynamic PDF Upload
- Conversational AI Chat Interface
- Retrieval-Augmented Generation (RAG)
- Semantic Search with FAISS
- Conversational Memory
- Source Citations & Page References
- Streamlit Web Application
- Groq LLM Integration

---

## 🛠 Tech Stack

- Python
- LangChain
- FAISS
- Groq API
- HuggingFace Embeddings
- Streamlit
- PyPDFLoader

---

## 🧠 Architecture

PDF Upload
→ Document Loading
→ Text Chunking
→ Embedding Generation
→ FAISS Vector Store
→ Similarity Retrieval
→ Prompt Engineering
→ LLM Response Generation

---

## 📌 Use Cases

- Financial Document Intelligence
- Loan Report Analysis
- Risk Assessment Assistance
- Enterprise Knowledge Retrieval

---

## 🚀 Deployment

Deployed using Streamlit Cloud.
